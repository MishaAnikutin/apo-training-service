# Запуск

Для начала нужно запустить docker-compose файл с БД

```bash
docker-compose up
```

После чего запустить бота
```shell
python -m bot
```

## Почему не Docker?
1. Новичкам проще понять код
2. Нет DevOps
3. Накладные расходы на память, скорость сборки и тестирования
