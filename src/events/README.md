# Ивенты

Архитектура веб приложения Event-Driven, поэтому тут для каждого события есть свой ивент:
- __register__ для формы регистрации (в боте)
- __auth__ для аторизации (на сайте)
- __start__ для меню с началом (в боте)
- __train__ для тренировки
- __statistics__ для получения статистики пользователя
- __train_filters__ для изменения фильтров

Для каждого ивента есть свой набор необходимых фичей: валидация запросов, клавиатуры, состояния.

Это сделано из-за того, что некоторые ивенты требуют специфической логики, которых больше нигде не будет.
